Gerenciador de Tarefas e Metas

Um aplicativo prático e interativo de linha de comando (CLI) desenvolvido em Node.js para ajudar no gerenciamento e acompanhamento de metas diárias. 

Este projeto foi construído e guiado a partir dos ensinamentos adquiridos durante o evento/curso da Rocketseat, servindo como uma excelente base para colocar em prática os fundamentos do JavaScript no ecossistema Node.

Funcionalidades

A aplicação roda direto no terminal e oferece um menu interativo onde é possível:
- Cadastrar: novas metas personalizadas.
- Listar: todas as metas abertas e concluídas.
- Marcar e desmarcar: o status de cada meta de forma visual usando o teclado.
- Persistência de dados: (opcional, dependendo de como você salvar as metas) para não perder o progresso.

Tecnologias Utilizadas

- JavaScript / Node.js
- @inquirer/prompts: Biblioteca responsável por criar as interfaces interativas, menus e caixas de seleção direto no terminal.

Pensando no Futuro (Próximos Passos)

Este projeto de terminal é apenas a primeira etapa. A lógica de negócios, as estruturas de repetição e o gerenciamento de estado construídos aqui servem como um "motor" sólido que pode ser expandido. 

Meus planos para a evolução deste sistema incluem:
1. Transformar em uma API Rest: Separar a lógica do terminal e criar rotas web.
2. Evoluir para a Web: Criar uma interface gráfica (Front-end) utilizando frameworks como React, permitindo o acesso pelo navegador.
3. Aplicativo: Utilizar a mesma base lógica para construir um aplicativo, levando o gerenciador de metas para os usuário de forma mais prática.
