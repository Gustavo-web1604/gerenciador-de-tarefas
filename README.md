# Gerenciador de Tarefas e Metas

Um aplicativo de linha de comando (CLI) feito em Node.js para ajudar no gerenciamento e acompanhamento de metas diárias.

Esse projeto foi construído a partir dos ensinamentos do evento/curso da Rocketseat, e serviu como uma ótima forma de colocar em prática os fundamentos do JavaScript no ecossistema Node.

## Funcionalidades

A aplicação roda direto no terminal e oferece um menu interativo onde é possível:

- Cadastrar novas metas.
- Listar todas as metas, abertas e concluídas.
- Marcar e desmarcar o status de cada meta usando o teclado.
- Persistência de dados, para não perder o progresso entre uma execução e outra.

## Tecnologias Utilizadas

- JavaScript / Node.js
- @inquirer/prompts: biblioteca usada para criar os menus e caixas de seleção interativas no terminal.

## Pensando no Futuro (Próximos Passos)

Esse projeto de terminal é só a primeira etapa. A lógica de negócios e o gerenciamento de estado construídos aqui servem como uma base sólida para expandir o sistema. Meus planos para a evolução incluem:

1. Transformar em uma API REST, separando a lógica do terminal e criando rotas web.
2. Evoluir para a Web, criando uma interface gráfica com frameworks como React.
3. Construir um aplicativo, usando essa mesma base lógica para levar o gerenciador de metas para os usuários de forma mais prática.
